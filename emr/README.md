# Personal Health Record

Static multilingual personal health record demo.

## Features

- Page-based browsing for six health record sections
- Five languages: Traditional Chinese, English, Japanese, Korean, Vietnamese
- Theme switcher: mint, rose, sand
- Cartoon profile illustration built into the page
- QR Code generator block on the final page
- Personal data stored in `data.js`

## Usage

Open `index.html` directly in a browser, or serve the folder with any static file server.

## Data

Replace the test data in `data.js` with your own structure-compatible content.
