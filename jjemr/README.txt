# Travel Medical Record (Offline)
- Open `index.html` directly (double-click). No server needed.
- Edit `data.js` to update your data.
- Replace photo: put your image at `assets/profile.png` (same filename) or change `person.photo_path` in `data.js`.

## Privacy
This is intended for offline use. If you publish to GitHub Pages, treat it as public data and mask/remove sensitive fields.
